# Cloudflare Worker 部署

文件：`worker.js`

---

## 部署方法

1. 打开 **Cloudflare → Workers & Pages**
2. 点击 **Create Worker**
3. 删除默认原始代码
4. 复制或粘贴 `worker.js` 内容
5. 点击 `Save and Deploy`  保存

---

## 使用

部署完成后，会生成一个内部地址：
或者绑定域名可公开访问地址
